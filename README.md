## 简介

光云工具箱(GMToolBox)

是我在多年前的一个空闲的时候使用batch/vbs/python写的一个Windows 平台下
多功能工具箱

可以帮助你更好的维护你的电脑，也有许多很有趣的功能


## 关于更新

按计划来说，本项目应该不会再更新
在几年前，我只是顺手写下这些代码，纯凭爱好
软件没有多少人用 现在的也被生活磨没了这份爱好

原本我是想使用PowerShell替代掉落后的Batch，
试用了 发现有些语句不兼容 要改的话太麻烦

后来又尝试了用Windows UI，
也以失败告终
